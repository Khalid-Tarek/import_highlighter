Import Error Highlighter

Copy this — Symptom

Symptom

A Maximo import preview reports BMXAA5598E errors with messages like:

  • CTGCC1046I — The CIRelation has not been saved, because the CI must exist before you can save the CIRelation.
  • BMXAA4129E — The record for Source=[CI], Target=[CI], Relation=[REL] already exists.

Scrolling between the preview text and the CSV to find which rows failed is tedious and error-prone, especially on large imports with hundreds of errors.

Copy this — Resolution

Resolution

Use the Import Error Highlighter tool:

  1. Paste the entire import preview text (including error messages) into the tool
  2. Select the CSV file you were trying to import
  3. Click Mark Import Errors

The tool parses every BMXAA5598E line, reads the affected CSV line ranges, and writes each error message into a new Import_Error column on the matching row. A .bak backup of the original CSV is saved automatically.
